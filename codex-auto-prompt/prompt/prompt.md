### 3. 验证与验收 (Verification & Review)
*   **单元测试/自测：** 执行代码并对照 `Acceptance Criteria` 进行逐条验证。

### 4. 状态更新与提交 (Completion & Commit)

*   **Git 提交：** 执行 Git 操作：
